# Happy Dice AI 🎲

一个超级可爱、支持 AI 叙事的 3D 骰子应用。

## 功能特点

- ✨ **可爱画风**：柔和的配色与弹力动画。
- 🤖 **AI 叙事**：集成 Google Gemini API，根据点数生成有趣的解说。
- 📱 **完美移动端适配**：支持微信内打开，防回弹，防误触。
- 📳 **摇一摇**：支持手机摇一摇投掷骰子。
- 🚀 **Vite + React**：极速启动与打包。

## 如何运行

1. 克隆项目
2. 安装依赖: `npm install`
3. 创建 `.env` 文件并填入你的 API Key: `API_KEY=你的GeminiKey`
4. 启动: `npm run dev`

## 部署

本项目配置了 Vercel 部署预设。
上传到 GitHub 后，在 Vercel 中导入即可。记得在 Vercel 后台配置环境变量 `API_KEY`。
